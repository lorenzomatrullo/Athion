## Athion

Athion is a clean, focused workout app designed to plan sessions quickly and log sets without friction. It emphasizes a polished dark UI, smooth inline editing, and reliable data storage—locally or in the cloud.

<br>

## Features
- Create and customize workout sessions with a clear, modern interface
- Add, edit, and remove exercises with sets and rep ranges
- Drag-and-drop reordering of exercises in both creation and edit flows
- Session Overview with inline edit mode to avoid disruptive navigation
- Start a workout to log weight (kg) and reps for each set
- Per-set “Done” action with previous workout values as ghost prompts
- SwiftData persistence; CloudKit sync when signed in with Apple; local-only when skipped
- Live switching between local and CloudKit stores without app relaunch
- Global keyboard dismiss toolbar for consistent input control
- Glassmorphic cards and balanced spacing for a cohesive visual style
- Onboarding with Sign in with Apple; Profile view with logged-in card and sign-out
- Analytics screen placeholder for future insights
